Erreur Corrigée

Fatal error: Class "OffreEmploi" not found

Erreur Identifiée

Message d'erreur :

Fatal error: Uncaught Error: Class "OffreEmploi" not found in C:\xampp\htdocs\document\digidocs\jobs\create.php:12

Cause :

  • La classe OffreEmploi n'existe pas dans le système
  • Le nom correct de la classe est Job
  • La méthode correcte est createOffre() et non create()

Corrections Apportées

1. Correction du nom de classe

Avant :

$offre = new OffreEmploi();

Après :

$offre = new Job();
2. Correction du nom de méthode

Avant :

$result = $offre->create($data);

Après :

$result = $offre->createOffre($data);
3. Suppression du paramètre inutile

Avant :

'utilisateur_id' => $_SESSION['user_id']

Après :

// Supprimé car la classe Job utilise directement $_SESSION['user_id']

Informations sur la Classe Job

Fichier : /classes/Job.php

Méthodes principales :

  • createOffre($data) - Créer une offre d'emploi
  • getOffres($filters) - Récupérer les offres
  • candidater($offreId, $candidatureData, $cvFile) - Postuler à une offre

Table utilisée : offres_emploi
