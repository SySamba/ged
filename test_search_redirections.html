Différences entre les Pages

documents/list.php
  • Affiche tous les documents de l'utilisateur
  • Pagination des résultats
  • Filtres de base (recherche, catégorie, dates)
  • Formulaire redirige maintenant vers search.php
documents/search.php
  • Interface de recherche avancée
  • Filtres étendus (taille, type MIME, utilisateur)
  • Résultats de recherche uniquement
  • Page dédiée à la recherche
