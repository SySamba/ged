Instructions de Test

Pour tester la redirection d'upload :
  1. Cliquez sur "Uploader un Document"
  2. Sélectionnez un fichier (PDF, Word, Excel, Image)
  3. Remplissez les informations optionnelles
  4. Cliquez sur "Uploader le document"
  5. Vérifiez que vous êtes redirigé vers la liste des documents
  6. Le message de succès doit s'afficher
Pour tester la création d'emploi :
  1. Cliquez sur "Créer une Offre d'Emploi"
  2. Remplissez le formulaire
  3. Cliquez sur "Créer l'Offre"
  4. Vérifiez la redirection vers la liste des emplois
