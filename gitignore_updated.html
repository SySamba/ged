Instructions de Configuration

Pour configurer un nouveau déploiement :

  1. Cloner le projet depuis Git
  2. Copier le fichier d'exemple :
    cp config/database.example.php config/database.php
  3. Modifier config/database.php avec vos paramètres :
    define('DB_HOST', 'localhost'); define('DB_NAME', 'votre_base'); define('DB_USER', 'votre_utilisateur'); define('DB_PASS', 'votre_mot_de_passe');
  4. Créer les dossiers nécessaires :
    mkdir uploads logs cache
Sécurité

Le fichier config/database.php contient vos mots de passe de base de données et ne sera jamais versionné dans Git. Seul le fichier d'exemple sera partagé.

Avantages
  • Tous les autres fichiers restent versionnés
  • Les scripts de base de données sont accessibles
  • Seules les informations sensibles sont protégées
  • Configuration plus flexible pour l'équipe
